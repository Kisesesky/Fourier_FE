# 2025-11-17 작업 요약

## 주요 변경
- Activities 뷰를 `lib/mocks/activities` 데이터로 재구성하여 날짜·멤버별 활동을 타임라인 형태로 표현했습니다.
- Recently Visited 화면에 grid/list 토글과 카드형 UI를 적용해 방문 기록을 다양한 레이아웃으로 확인할 수 있습니다.
- 멤버 초대 플로우를 `InviteMemberModal` 모달로 전환하고, `@/workspace/members/_model/mocks` 데이터를 사용하는 Members 뷰 UX를 개선했습니다.
- `app/page.tsx`의 방대한 JSX를 `components/workspace/*`, `mocks/workspace.ts`, `types/workspace.ts`, `hooks/useWorkspaceUser.ts`로 모듈화했습니다.
- WorkspaceTopbar를 브라우저 탭과 비슷한 구조로 바꾸어 Home 아이콘+텍스트, 탭 닫기(X), Recently closed 복원 메뉴를 제공합니다.
- LeftNav가 팀 선택, 즐겨찾기·최근 버튼(카운트 표시) 및 탭 전환을 지원하고, “My Favorites” 뷰가 실제 즐겨찾기 프로젝트를 grid/list로 보여줍니다.
- ProjectCard(grid)는 고정 높이 카드로 통일하여 레이아웃이 균일하게 유지됩니다.

## 세부 작업
- `app/page.tsx`에서 즐겨찾기 뷰와 최근 방문 뷰 모두 grid/list 전환 버튼을 추가하고, 탭별 상태를 정리했습니다.
- `components/workspace/LeftNav.tsx`에 `onSelectTeam`, `favoriteCount`, `recentCount`를 도입해 UX를 강화했습니다.
- `components/workspace/views/RecentVisitedView.tsx`를 grid/list 토글이 가능한 새 컴포넌트로 작성했습니다.
- `components/workspace/projects/ProjectCard.tsx` grid 버전을 고정 높이 카드로 재작성하고 상단 메타(소유자/업데이트 시간)를 제거했습니다.
- `pages/_document.tsx`를 App Router에서도 안정적으로 로드되도록 간단한 함수 컴포넌트로 유지했습니다.
- 모든 변경 후 `npm run build`로 Next.js 빌드/타입 검사를 통과했습니다.
