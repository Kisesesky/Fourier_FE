- 캘린더를 개별 페이지로 분리해 `/calendar/[calendarId]` 전용 뷰를 추가하고, 해당 캘린더만 필터링해서 보여주도록 `CalendarView`를 스코프 처리함.
- 캘린더 생성 시 자동으로 새 캘린더 페이지로 이동하도록 제출 플로우를 변경하고, 헤더에 “다른 캘린더로 이동” 드롭다운을 추가함.
- 사이드 패널 캘린더 목록에 “전체 캘린더” 버튼과 개별 캘린더 이동 버튼을 추가하고, 관리/삭제(확인 포함)를 바로 수행하도록 연결함.
- `CalendarView`의 `events` 참조 오류를 수정하고, 포커스 캘린더에 맞게 Draft 기본 캘린더를 동기화함.
- 백엔드에서 프로젝트 캘린더 카테고리가 없을 경우 기본 캘린더를 생성해 반환하도록 `calendar.service.ts`를 보완함.
